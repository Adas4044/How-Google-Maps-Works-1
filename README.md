## Installation
1. Clone repository to your local machine
2. Open terminal and `cd` into repository folder
3. Run `npm install`
4. Run `vite` or `npm run dev`

## Available algorithms
- BFS
- DFS
- Bidirectional BFS
- Greedy
- A* algorithm
- Bidirectional A*
- Bidirectional A* with lookup table
